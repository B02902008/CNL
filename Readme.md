# Bomb.io Game

## How to Run

1. Clone the repository

```
git clone https://github.com/b02902008/CNL.git
```

2. Change directory to cloned project

```
cd CNL
```

3. Install dependency packages

```
npm install
```

4. Run server

```
npm start
```

- Note that lastest NodeJs may cause error, use NodeJs 9.X.X
